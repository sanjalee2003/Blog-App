# PHP Blog App

This is a blog application built using **PHP**, **HTML**, **CSS**, and **JavaScript**.

---

## Functionalities

### 1. User Authentication & Authorization
- Users can **register**, **log in**, and **log out**.
- Only authenticated users can **create**, **update**, or **delete** their own blogs.
- Authorization ensures that a user **cannot edit or delete** blogs created by other users.

---

### 2. Blog Management
Users are able to:
- **Add new blogs** using a blog editor (SimpleMDE is used).
- **Read blogs** â€” all blogs are displayed on the home page.
- **Update** their own blogs.
- **Delete** their own blogs.

---

### 3. Frontend Features
- **Home Page:** Displays a list of all blogs.
- **Single Blog View Page:** Shows blog content, author, and date.
- **Blog Editor Page:** For creating and updating blogs.
- **Responsive & Clean UI:** Built with HTML, CSS, and JavaScript.
